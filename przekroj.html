Poniszy fragment kodu definiuje klas o nazwie "Plaster". Klasa ta ma kilka metod, ktre wykonuj rne operacje na punktach plastra.

Metoda init jest konstruktorem klasy. Przyjmuje ona dwa argumenty: points, ktry reprezentuje punkty na plastrze, i plaster_number, ktry reprezentuje numer tego plastra.

Metoda get_num_points() zwraca liczb punktw na plastrze, uywajc funkcji wbudowanej w pythonie funkcji len().

Metoda interpolate_circle() suy do interpolacji okrgu na podstawie punktw na plastrze. Jeli liczba punktw jest mniejsza ni 3, metoda zwraca None, None. Nastpnie, metoda tworzy zmienne x, y, z na podstawie wartoci w punktach. Wykorzystuje one dane z punktw, gdzie x to wsprzdna x, y to wsprzdna y, a z to wsprzdna z. Funkcja objective_function() jest zagniedon funkcj w metodzie interpolate_circle, ktra przyjmuje argument center_radius, ktry reprezentuje rodek okrgu i promie. Oblicza ona odlego kadego punktu od tego okrgu i zwraca sum kwadratw tych odlegoci. Nastpnie tworzona jest lista center_radius_guess, ktra zawiera przyblione wartoci dla rodka okrgu i promienia. rodek okrgu to rednia warto wsprzdnych x i y, a promie to odchylenie standardowe wartoci x. Nastpnie wywoywana jest funkcja minimize, ktra minimalizuje warto funkcji objective_function dla danego przyblienia center_radius_guess.

Funkcja minimize uywa metody Neldera-Meada. Metoda ta (znana rwnie jako metoda sympleksu, metoda ameby lub metoda wielocianu) to specjalny sposb szukania najwyszego lub najniszego punktu w problemach matematycznych z wieloma zmiennymi.

Jak dziaa ta metoda?: Wyobramy sobie pagrkowaty krajobraz z garbami i dolinami. Zaczynamy od zestawu punktw na krajobrazie. Kady punkt reprezentuje "domys", gdzie moe znajdowa si najwyszy lub najniszy punkt. Te punkty s jak zespoy badawcze. Badacze poruszaj si, podejmujc kroki w gr lub w d pagrkw. Zapisuj, ktry krok zblia ich do najwyszego lub najniszego punktu. Jeli znajd lepsze miejsce, przenosz si tam i kontynuuj poszukiwania. Robi to tak dugo, a nie bd mogli znale adnego lepszego miejsca do przejcia. Wtedy wiedz, e znaleli najwyszy lub najniszy punkt.

Zatem metoda Neldera-Meada to inteligentny sposb eksplorowania krajobrazu i znajdowania najniszego lub najwyszego punktu. Pomaga rozwizywa trudne problemy w matematyce lub naukach przyrodniczych, gdzie celem jest znalezienie jednego najlepszego rozwizania.

Wynik tej minimalizacji jest przechowywany w zmiennej result. Nastpnie, z wyniku result pobierane s wartoci dla rodka okrgu (center) oraz promienia (radius). Dodatkowo obliczane jest odchylenie (deviation) na podstawie wartoci minimalnej funkcji i liczby punktw.

Na koniec, metoda interpolate_circle zwraca obliczone odchylenie i rodek okrgu w postaci listy deviation, center.

class Plaster:
    def __init__(self, points, plaster_number):
        self.points = points
        self.plaster_number = plaster_number

    def get_num_points(self):
        return len(self.points)

    def interpolate_circle(self):
        if len(self.points) < 3:
            return None, None

        x = self.points["x"]
        y = self.points["y"]
        z = self.points["z"]

        def objective_function(center_radius):
            cx, cy, r = center_radius
            distance = np.sqrt((x - cx) ** 2 + (y - cy) ** 2)
            return np.sum((distance - r) ** 2)

        center_radius_guess = [np.mean(x), np.mean(y), np.std(x)]


        result = minimize(objective_function, center_radius_guess, method='Nelder-Mead')
        center = result.x.tolist()
        center.append(np.mean(z))
        radius = result.x[2]
        deviation = np.sqrt(result.fun / len(self.points))

        return deviation, center

Nastpnie, metoda oblicza minimaln (min_z) i maksymaln (max_z) warto wsprzdnej z w punktach chmury. Oblicza rwnie zakres wartoci z jako rnic midzy maksymaln i minimaln wartoci. Nastpnie oblicza rozmiar kroku (step_size) jako podzia zakresu wartoci z przez liczb plasterw.

Obiekt klasy Plaster jest dodawany do listy plasters.

class PointCloud:
    def __init__(self, filename):
        self.filename = filename
        self.points = None

    def load_point_cloud(self):
        try:
            point_cloud = laspy.read(self.filename)
            self.points = point_cloud.points
        except Exception as e:
            print(f"Error loading point cloud: {e}")

    def divide_into_plasters(self, num_plasters):
        if self.points is None:
            print("Point cloud not loaded.")
            return []

        min_z = np.min(self.points["z"])
        max_z = np.max(self.points["z"])
        z_range = max_z - min_z
        step_size = z_range / num_plasters

        plasters = []
        for i in range(num_plasters):
            min_z_plaster = min_z + i * step_size
            max_z_plaster = min_z + (i + 1) * step_size
            plaster_points = self.points[(self.points["z"] >= min_z_plaster) & (self.points["z"] < max_z_plaster)]
            plaster = Plaster(plaster_points, i + 1)
            plasters.append(plaster)

        return plasters

point_cloud = PointCloud("Komin.las")
point_cloud.load_point_cloud()

num_plasters = 10

plasters = point_cloud.divide_into_plasters(num_plasters)
centers = []
deviations = []

for plaster in plasters:
    deviation, center = plaster.interpolate_circle()
    if deviation is not None:
        centers.append(center)
        deviations.append(deviation)

centers = np.array(centers)
